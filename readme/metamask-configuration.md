# Metamask configuration

Follow instructions [from the official Metamask](https://support.metamask.io/configure/networks/how-to-add-a-custom-network-rpc#adding-a-network-manually) documentation under the "Adding a network manually" section, and use the following settings:

* Network name: `Taraxa Mainnet`&#x20;
* Default RPC URL: `https://mainnet.rpc.tara.community`&#x20;
* Chain ID: `841`&#x20;
* Currency symbol: `TARA`&#x20;
* Block explorer URL: `https://mainnet.explorer.tara.community`

Please note that the block explorer is not fully operational so you can also leave it blank if Metamask complains about it.
